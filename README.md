# Room8
A full-stack roommate management app that simplifies shared living — track expenses, manage chores, and coordinate household responsibilities in one place.

---

## ✨ Highlights
- 🏠 Replaces scattered group chats and spreadsheets with a unified household system  
- 💸 Creates accountability through clear expense and responsibility tracking  
- 🧩 Built with a clean frontend–backend separation to support future scaling

---

## 🛠 Tech Stack

- Swift
- SwiftUI
- MVVM architecture
- Modular frontend components
- Backend designed for scalable multi-user sync

---

## 🚀 Getting Started

```bash
git clone https://github.com/fatmaayad13/Room8.git
open Room8.xcodeproj
```

Requirements:
- macOS
- Xcode 15+
- iOS 17+

See the [Setup Guide](Room8App/SETUP.md) for full configuration instructions.
