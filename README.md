# Room8
A full-stack roommate management app that simplifies shared living — track expenses, manage chores, and coordinate household responsibilities in one place.

---

## ✨ Highlights

- 📱 SwiftUI-based iOS app
- 💸 Shared expense tracking with balance calculations
- 🧹 Chore assignment & completion tracking
- 🧩 Clean separation of frontend and backend architecture
- 🚀 Designed for future App Store deployment

---

## 🛠 Tech Stack

- Swift
- SwiftUI
- MVVM architecture
- Modular frontend components
- Backend designed for scalable multi-user sync

---

## 🚀 Getting Started

```bash
git clone https://github.com/fatmaayad13/Room8.git
open Room8.xcodeproj
```

Requirements:
- macOS
- Xcode 15+
- iOS 17+
